# EjemplosCodigo2025
Repositorio con ejemplos de código escritos en C#. (desde la versión 8)

Este repositorio cuenta con una aplicación de consola en .net, en donde la misma aplicación de consola permitirá al usuario elegir que tareas ejecutar.

El código muestra ejemplos básicos del uso de C#.

## Ejemplos incluidos:
- Comparación inexacta de valores de variables tipo Double.
- Comparación valores de variables tipo Decimal.
